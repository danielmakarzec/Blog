# Ruby on Rails Blog

This project was created as a part of my learning process.
The goal was to practise the following aspects:

* MVC
* Polymorphic Models
* User authentication with Devise
* Active storage attachments with Cloudinary(SaaS)
* AJAX

## Demo

ror-bloger.herokuapp.com

## Built With

- Framework:
  - Ruby on rails
- Database:
  - postgresql
- Frontend librareies:
  - jQuery
  - Bootstrap
- Ruby gems:
  - see Gemfile
- Webpack dependencies:
  - see package.json

## Instalation

``` console
$ bundle
$ rails db:migrate db:seed
$ rails s
```

Open the browser and go to [localhost:3000](http://localhost:3000)

## Tests

``` console
$ rake spec
```
or
``` console
$ rails spec
```
